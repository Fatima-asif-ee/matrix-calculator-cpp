# Matrix Calculator

A Menu-driven console-based matrix calculator that performa basic calculations(addition, subtraction, multiplication).

## Features
- Input from user (no. of rows and columns and elements)
- Input validation to check compatibility using no. of rows and columns of both matrices
- Display matrices (A and B)
- Matrix Addition
- Matrix Subtraction
- Matrix multiplication
- A Menu to choose the operation
- Error handling for invalid choice

## How to Run

### Requirements
- Any c++ compiler
- Any IDE like VS Code(optional but preferable)
### Run
- Clone the repo or simply copy paste the code on your compile or type it
- Compile and run the code

## How To Use
- A menu appears on the console
- select the number for corresponding operation
- (1 to input matrix A
  2 to input matrix B
  3 to display them both
  similarly choose operation
  if program invalidates the inputs restart using correct dimensions for matrices for example equal no. of rows and columns of both matrices
  enter 0 to exit the program)

 ## known limitation
 - if only one matrix is entered by the user and user chooses to display it, the program still  displays two matrices with garbage values in other matrix. Enter both matrices before displaying to avoid garbage values.
